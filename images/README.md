# Images Folder

This folder contains images for your portfolio.

## Profile Picture

To add your profile picture:

1. **Prepare your image:**
   - Use a high-quality photo (at least 500x500 pixels)
   - Square aspect ratio works best
   - Good lighting and clear face visibility
   - Professional or semi-professional appearance

2. **File requirements:**
   - Name your file `profile.jpg` (or update the HTML if using a different name)
   - Supported formats: JPG, PNG, WebP
   - Recommended size: 500-1000px square
   - File size: Keep under 500KB for fast loading

3. **How to add:**
   - Copy your image file to this `images` folder
   - Rename it to `profile.jpg`
   - The website will automatically display it

## Project Images

You can also add project screenshots here:

Then update the HTML to reference these images in your project cards.

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Consider using WebP format for better compression
- Ensure images are properly sized before uploading
- Use descriptive alt text for accessibility
