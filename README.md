# Jumoke Joseph Ejirooghene - Portfolio

A modern, responsive personal portfolio website showcasing my skills, projects, and experience as a Software Engineer.

🌐 **Live Site**: [https://jumoke-joseph-portfolio.vercel.app](https://jumoke-joseph-portfolio.vercel.app)

## 🚀 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and gradient themes
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic navigation
- **Contact Form**: Functional contact form with custom success messages
- **Performance Optimized**: Fast loading with efficient CSS and JavaScript

## 📋 Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About**: Personal introduction with professional headshot
3. **Technical Expertise**: Skills organized by category (Frontend, Backend, Languages, etc.)
4. **Work Experience**: Professional timeline with detailed achievements
5. **Featured Projects**: Real projects with live demos and GitHub links
6. **Contact**: Contact information and functional contact form

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Custom CSS with Flexbox and Grid
- **Icons**: Font Awesome
- **Hosting**: Vercel
- **Version Control**: Git & GitHub

## 🎨 Color Scheme

- **Primary Gradient**: `#667eea` → `#764ba2` (Purple-blue to deep purple)
- **Accent**: `#fbbf24` (Golden yellow)
- **Text**: Various shades of gray for optimal readability

## 📱 Responsive Breakpoints

- **Desktop**: 1200px+
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px

## 🚀 Deployment

This portfolio is deployed on Vercel with automatic deployments from the main branch.

### Deploy Your Own

1. Fork this repository
2. Connect your GitHub account to Vercel
3. Import the project and deploy
4. Customize with your own information
